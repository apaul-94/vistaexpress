== Description ==
Vista Express is the ideal and best WordPress theme for individuals who value both elegance and utility. With its clean, modern style and emphasis on user-friendly customization, Vista Express enables you to create a gorgeous website without any coding experience. Whether you're creating a personal blog, a professional portfolio, or a company website, Vista Express can help you stand out with versatile style options, responsive design, and SEO optimization. Customizable headers, footers, fonts, and colors give you complete control over the appearance of your website.

== Installation ==
1. Download the theme from the WordPress repository or upload it manually.
2. Go to your WordPress Dashboard.
3. Navigate to Appearance > Themes.
4. Click on Add New and then Upload Theme.
5. Choose the downloaded `vistaexpress.zip` file and click Install Now.
6. After installation, click Activate to use the theme.

== Features ==
* Responsive Design: Looks great on any device.
* SEO Optimization: Built with SEO best practices in mind.
* Customizable Options: Headers, footers, fonts, and colors are fully customizable.
* Translation Ready: Supports multiple languages with RTL support.
* Multiple Layouts: Includes two-columns, right-sidebar, left-sidebar, and full-width templates.
* Widget Ready: Includes footer widgets and sidebar widget areas.
* Threaded Comments: Supports nested comments to enhance community interaction.
* Editor Style: Matches front-end styling for a seamless editing experience.
* Full-width Template: Create stunning pages with a full-width template.

== Customization ==
Vista Express includes a variety of customization options available under Appearance > Customize. Here, you can adjust:
- Site Identity: Change the site title, tagline, and logo.
- Post Page Settings: In this setting you can choose the limit of words in title & excerpt. Also you can choose the Read More button and change the Read More button label.
- New Post Scroll Section: Admin can enable and disable the Recent Posts Scroll Section from this settting.
- Footer Copyright Settings: If admin wants to change the Footer Copyright Text then he or she can do from this setting.
- Menus: In this setting admin will able to choose the menu which he or she want to display at the navbar.
- Widgets: Customize the Sidebar and Footer content as well.
- Home Settings: By this setting admin can choose the Home page and Blog Page as well. And also can able to choose the Latest Post page.
== Note: ==
In this version (V1.0.0), these customizations are only available. So, keep in touch with us. In future, we will add more customization options by which admin or user able to customize more things on this theme.

== Frequently Asked Questions ==

= Is Vista Express compatible with the latest version of WordPress? =
Yes, Vista Express is tested up to WordPress version 6.6.

= Can I use Vista Express for an e-commerce website? =
Absolutely! Vista Express is fully compatible with WooCommerce, making it an excellent choice for online stores.

= Does Vista Express support RTL languages? =
Yes, Vista Express includes RTL language support for right-to-left languages.

Vista Express includes a variety of customization options that can be accessed via **Appearance > Customize** in your WordPress Dashboard.
